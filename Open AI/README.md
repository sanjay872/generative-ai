# Open AI

Open AI website.
It can be used to do prompts, assistance, TTS... etc.

## Software requires
- Anaconda
- VS code
- Git

## Setup
- Open anaconda cmd

- Create environment
```bash
conda create -n openaidemo python=3.10 -y
```
- Activate environment
```bash
conda activate openaidemo
```
- Install packages
```bash
pip install -r requirements.txt
```
